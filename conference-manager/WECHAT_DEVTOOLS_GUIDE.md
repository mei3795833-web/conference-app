# 微信开发者工具使用指南

## 1. 下载微信开发者工具

### 官方下载地址
访问微信开发者工具官方下载页面：
**https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html**

### 选择版本
- **Windows** → 下载 `.exe` 安装包（推荐稳定版 Stable Build）
- **macOS** → 下载 `.dmg` 安装包

### 安装步骤
1. 双击下载的安装包
2. 点击"下一步" → "我接受"许可协议
3. 选择安装路径（建议非系统盘，如 D 盘）
4. 等待安装完成，点击"完成"

---

## 2. 导入项目

### 步骤 1：打开微信开发者工具
- 桌面或开始菜单找到"微信开发者工具"图标
- 双击打开，首次使用需要微信扫码登录

### 步骤 2：导入项目
1. 点击左侧 **"导入项目"** 按钮（或点击 + 号）
2. 在弹出的文件选择器中，找到项目目录：
   ```
   C:\Users\bever\.qclaw\workspace-agent-240d4148\conference-manager\
   ```
3. 选择该文件夹，点击"选择文件夹"

### 步骤 3：配置项目信息
导入后会弹出项目配置窗口：

| 配置项 | 填写内容 |
|--------|----------|
| **项目名称** | 大会管理系统（可自定义） |
| **目录** | 自动填充（刚才选择的路径） |
| **AppID** | 见下方说明 |
| **开发模式** | 选择"小程序" |

## 3. 配置 AppID

### 方式一：使用测试号（快速体验）
- 在 AppID 输入框下方点击 **"测试号"**
- 系统会自动生成一个测试用的 AppID
- 适合本地开发和功能预览

### 方式二：使用正式 AppID（推荐）
1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 注册/登录小程序账号
3. 进入"开发" → "开发管理" → "开发设置"
4. 找到 **AppID(小程序ID)**，复制该字符串
5. 粘贴到微信开发者工具的 AppID 输入框

> ⚠️ **注意**：正式 AppID 需要完成小程序注册和认证才能获取

## 4. 预览与调试

### 编译运行
- 点击工具栏的 **"编译"** 按钮（或按 Ctrl+S 保存自动编译）
- 左侧模拟器会实时显示小程序界面

### 真机预览
1. 点击工具栏的 **"预览"** 按钮
2. 生成二维码
3. 用微信扫码即可在手机上预览

### 调试功能
- **Console**：查看日志输出
- **Network**：查看网络请求
- **Storage**：查看本地存储
- **Wxml**：查看页面结构
- **Sources**：查看源码和断点调试

## 6. 常见问题

### Q1: 导入后提示"找不到 app.json"？
**解决**：确保选择的是 `conference-manager` 文件夹，而不是上级目录

### Q2: 提示"AppID 不合法"？
**解决**：检查 AppID 是否复制完整（通常是 18 位字符串），或先使用测试号

### Q3: 地图组件不显示？
**解决**：需要在微信公众平台配置地图 key，并在 `app.json` 中配置权限

### Q4: 如何添加图片资源？
**解决**：将图片文件放入 `images/` 目录，代码中引用路径如 `/images/home.png`
