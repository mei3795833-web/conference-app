# 大会管理系统小程序

## 项目概述
专为大型会议活动提供一站式管理服务的小程序，包含大会指挥、交通指导、住宿安排等核心功能。

## 功能模块

### 1. 首页
- 大会信息展示
- 快捷入口（议程、人员、通知）
- 今日议程时间轴
- 实时数据统计

### 2. 指挥模块
- 指挥状态看板
- 任务管理（紧急/重要/普通）
- 人员分布统计
- 实时消息
- 紧急处理

### 3. 交通模块
- 地图导航
- 交通方式选择（公共交通/打车/自驾/接驳车）
- 实时路况
- 停车信息
- 紧急联系

### 4. 住宿模块
- 我的住宿信息
- 酒店列表（支持筛选）
- 住宿统计
- 入住须知

### 5. 个人中心
- 用户信息
- 角色切换
- 我的议程/住宿/交通
- 通知消息
- 隐私协议

## 技术栈
- 微信小程序原生开发
- WXML + WXSS + JavaScript
- 微信云开发（可选）


## 使用说明

### 开发环境
1. 下载微信开发者工具
2. 导入项目
3. 配置 AppID

### 部署上线
1. 在微信公众平台注册小程序
2. 配置服务器域名
3. 上传代码
4. 提交审核

## 审核注意事项
- 已配置隐私协议页面
- 已申请位置权限（scope.userLocation）
- 符合微信小程序运营规范
